# Skill: Content Generation

## Skill ID
skill_content_generation

## Purpose
Generate content drafts from an approved trend.

This skill produces text ONLY.
It does NOT approve content.
It does NOT publish content.

---

## Allowed Callers
- Worker Agent

---

## Preconditions

- Trend MUST be approved
- Influencer MUST be Active
- Input schema MUST validate

---

## Postconditions

- Content draft is returned
- Risk classification is attached
